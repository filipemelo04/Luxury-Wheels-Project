# Luxury-Wheels-Project
O projeto Luxury Wheels é uma aplicação desktop desenvolvida em Python para a gestão completa de uma empresa de aluguer de veículos.
A aplicação permite ao utilizador gerir veículos, clientes, reservas e pagamentos de forma integrada, com uma interface intuitiva e funcional.
Entre as principais funcionalidades incluem-se:
- Dashboard interativo com visualização de reservas ativas, novos clientes e veículos disponíveis;
- Gestão de veículos (adição, edição, eliminação e exportação para Excel);
- Acompanhamento automático das datas de revisão e inspeção dos veículos;
- Filtros de pesquisa avançados (tipo, categoria, transmissão, etc.);
- Gráficos analíticos sobre ocupação, receitas diárias e tipo de veículo, utilizando Matplotlib;
- Arrastar e largar de imagens para associar fotos aos veículos (drag-and-drop);
- Exportação de relatórios em formato Excel com Pandas;
- Base de dados local SQLite, garantindo persistência dos dados;
- Interface construída com Tkinter e TkinterDnD2, proporcionando uma experiência de utilizador moderna e responsiva.

⚙️ Tecnologias e Bibliotecas Utilizadas:
- Python 3
- Tkinter e TkinterDnD2 → Interface gráfica e funcionalidade drag-and-drop
- SQLite3 → Base de dados local
- tkcalendar → Seleção de datas com interface gráfica
- Pandas → Exportação e manipulação de dados
- Matplotlib → Geração de gráficos e análises visuais
- NumPy → Cálculos e regressões lineares
- Schedule / Threading → Execução de tarefas agendadas e paralelas
- OS / Shutil → Gestão de ficheiros e diretórios
